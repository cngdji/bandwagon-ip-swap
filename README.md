## 搬瓦工付费换 IP 是什么：先搞清楚一件事

搬瓦工 VPS 每台分配一个独立 IPv4，这个 IP 只有你在用。问题是，一旦这个 IP 被大陆防火墙列入黑名单，VPS 本身没问题，但所有来自国内的连接都会被切断。

换 IP 就是重新分配一个干净的 IPv4 给你的 VPS，旧 IP 被回收掉。**这是目前修复被封 IP 最彻底的办法**，没有之一。

## 先确认 IP 真的被封了

换 IP 之前，先做一步检查——确认问题出在 IP 上，而不是端口被封或者本地网络的问题。

搬瓦工后台提供了官方检测工具。操作路径如下：

1. 登录搬瓦工客户后台
2. 进入 Services → My Services，找到要检测的 VPS
3. 点击 **KiwiVM Control Panel** 进入 KiwiVM 面板
4. 在 KiwiVM 面板打开 IP 检测页（在 KiwiVM 内的 Main Controls 区域）
5. 点击 **Test Main IP** 按钮

结果只有两种：

- **IP NOT BLOCKED** → IP 没问题，连不上是别的原因，不需要换 IP
- **IP BLOCKED** → IP 已被封，可以继续下一步

这一步别跳过。付了钱换 IP，结果发现其实是端口问题，这种亏不少人吃过。

## 换完 IP 还是用不了？先别急着开工单

付完钱、IP 换好了，但连接还是失败——这种情况是有的，原因通常是 KiwiVM 的 DHCP 服务器还没刷新，返回的还是旧 IP 信息。

解决方法：

- 在 KiwiVM 面板里停止 VPS（Stop），等几秒，再重新启动（Start）
- 重新连接，不要用缓存的旧 IP，从 KiwiVM 面板重新确认 IP 地址后再填

如果这样也不行，直接回复工单告诉客服，毕竟你付钱换的 IP，客服有义务保证新 IP 是可用的。
